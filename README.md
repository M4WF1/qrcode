# QR Code Generator
This is a simple QR Code Generator tool. It is for both Termux and Linux users.

## Commands
```
pkg update
pkg upgrade
pkg install git
pkg install python
rm -rf qrcode
git clone https://github.com/mawfii/qrcode
cd qrcode
python qrcode.py
```
